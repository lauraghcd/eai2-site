# Product Review Knowledge Base

Last updated: 2026-04-30

## Product Position

EAI2 should read less like a generic blog and more like an independent technical field guide. The strongest current wedge is not news velocity; it is structured, cross-linked explanations for engineers, researchers, and strategy readers entering embodied AI.

## Current Product Truth

- Static Hugo architecture is fast and cheap.
- Core EEAT repairs have been implemented: author page, article schema, breadcrumbs, TL;DR blocks, and deeper internal links.
- The main remaining conversion weakness is end-of-article dead ends.
- The main learning weakness is that the Learning Path content exists but does not yet feel like a guided course.

## Competitive Pressure

- The Robot Report and IEEE Spectrum win on authority and news flow.
- Vendor docs win on first-party implementation detail.
- EAI2 can win by being vendor-neutral, structured, and easier to follow across the whole embodied AI stack.
